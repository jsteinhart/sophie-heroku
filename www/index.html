Welcome to SoPHIE

Please enable mod_rewrite to use the Software Platform for Human Interaction Experiments:

To activate mod_rewrite functionality please look into your Webserver specific documentation.

Procedure for XAMPP on Windows or Linux
  1. Locate the file httpd.conf containing the Apache configuration.
  2. Search for LoadModule rewrite_module modules/mod_rewrite.so. There should be a preceding hash (#) which needs to be removed.
  3. Restart the Apache webserver though the XAMPP console. You may use the cli command apache2ctl graceful on the console.
  4. Reload this page.
Procedure for Apache on a Debian Linux or Ubuntu Distribution
  1. Activate the module by executing the command a2enmod rewrite on the console with root priviledges.
  2. Restart the Apache webserver by running apache2ctl graceful on the console with root priviledges.
  3. Reload this page.
More information on the Apache Module mod_rewrite can be found on its homepage.
